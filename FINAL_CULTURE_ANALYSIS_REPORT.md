- **Takeaway**: Help employees see how their work matters

**5. Employee Voice Enhances Wellbeing (CTRL03)**
- Employees who feel heard and valued (CTRL03) show **47% higher wellbeing**
- Feeling valued drives engagement
- **Takeaway**: Create systems where employee input shapes decisions

## Technical Appendix

### Statistical Methods

This analysis employs multilevel hierarchical linear modeling to account for the nested structure of the data, where individual employees (Level 1) are nested within organizations (Level 2). This approach is statistically necessary because:

1. **Intraclass Correlation Coefficient (ICC) = 0.3870**: This indicates that 38.7% of variance in responses occurs between organizations, while 61.3% occurs within organizations. An ICC above 0.1 requires multilevel modeling to avoid biased estimates.

2. **Non-Independence of Observations**: Employees within the same organization share common context (leadership, policies, culture), making their responses non-independent. Standard regression techniques would violate the independence assumption and produce inflated Type I error rates.

3. **Model Specification**: The model uses a two-level structure:
   - **Level 1 (Individual)**: Employee responses to survey questions
   - **Level 2 (Organization)**: Organization-level characteristics and random intercepts

**Model Equation:**
```
Y_ij = γ_00 + γ_10 * X_ij + u_0j + r_ij
```

Where:
- `Y_ij` = Outcome for individual i in organization j
- `γ_00` = Grand mean intercept
- `γ_10` = Fixed effect slope (relationship between predictor and outcome)
- `X_ij` = Individual-level predictor
- `u_0j` = Random intercept for organization j (between-org variance: τ_00)
- `r_ij` = Individual-level residual (within-org variance: σ²)

**Variance Components:**
- **Between-Organization Variance (τ_00)**: Measures how much organizations differ from each other
- **Within-Organization Variance (σ²)**: Measures how much individuals differ within organizations
- **ICC = τ_00 / (τ_00 + σ²)**: Proportion of total variance that is between organizations

### Model Estimation

Models were estimated using:
- **Primary Method**: Mixed Linear Models (MixedLM) via statsmodels in Python
- **Fallback Method**: OLS with clustered standard errors when MixedLM unavailable
- **Estimation**: Maximum Likelihood (ML) or Restricted Maximum Likelihood (REML)
- **Validation**: Model diagnostics including residual analysis, variance component testing, and cross-validation

### Key Statistical Metrics

**Slope (β)**: The change in the outcome variable (e.g., Culture Health score) for each 1-point increase in the predictor variable. For example, a slope of 21.2 means that for every 1-point increase in Overall Culture Rating, Culture Health increases by 21.2 points.

**R-Squared (R²)**: The proportion of variance in the outcome explained by the predictor. R² values range from 0 to 1, with higher values indicating stronger predictive relationships:
- **R² = 0.636**: Overall Culture Rating explains 63.6% of variance in Culture Health
- **R² = 0.618**: Core Value Identification explains 61.8% of variance in Core Values metric
- **R² = 0.597**: Purpose & Value Identification explains 59.7% of variance in Motivation

**Marginal R²**: Variance explained by fixed effects (predictor variables) only  
**Conditional R²**: Variance explained by both fixed effects and random effects (organization-level variance)

**P-Values**: Statistical significance tests. All reported relationships are statistically significant (p < 0.001) unless otherwise noted.
